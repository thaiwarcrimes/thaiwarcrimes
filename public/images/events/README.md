# Event Images

This directory contains images for timeline events. Each event has its own subdirectory.

## Directory Structure

```
events/
├── preah-vihear-2008/
├── dang-rek-2011/
└── disputed-2013/
```

## Image Guidelines

- Use high-quality, verifiable images
- Include proper attribution and source information
- Recommended sizes:
  - Timeline thumbnails: 800x600px
  - Article hero images: 1200x800px
  - In-article images: 1000x750px
- Supported formats: JPG, PNG, WebP

## Placeholder Images

For development, you can use placeholder images from services like:
- https://placehold.co/
- https://picsum.photos/

Replace with actual documented images before production deployment.
